
# 🌐 Bootstrap Task 1 - Personal Profile Website

## 📌 Project Description
This project is a responsive personal profile website developed using Bootstrap 5.  
It includes multiple pages such as Home, About, and Contact.

---

## 🚀 Features
- Responsive Navbar with Hamburger Menu  
- Cards with Hover Animation  
- Progress Bars for Skills  
- Contact Form  
- Alerts and Buttons  
- Clean UI Design  

---

## 🛠️ Technologies Used
- HTML5  
- Bootstrap 5  
- CSS  

---

## 📂 Project Structure
Task1-Bootstrap/
│── index.html  
│── about.html  
│── contact.html
